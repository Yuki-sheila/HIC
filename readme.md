# FDGC论文复现代码与结果



## 运行
改变data的路径，然后运行main.py即可

> python main.py 

## 基本实现
###图像预处理：
- PCA降维 
- 切成patch_size为19的小图
- 数据集划分、转one-hot编码
- 标签平滑
- 数据可视化

###模型搭建
- 两个CNN分支
- 一个GCN分支
- 三路特征concat
- 最后经过几层fc层得到预测的结果

###评价指标
最好的一次结果保存在./Results/best_result.txt中
- OA：94.70%
- AA：91.47%
- Kappa：93.88
